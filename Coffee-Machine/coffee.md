# Coffee Machine Simulation

## Objective
Simulate a coffee machine using C++ classes and OOP principles.

## Concepts Demonstrated
- Classes & Objects
- Encapsulation with private members
- Constructors for initialization
- Member functions for operations
- Conditional logic for payments and resource checks

## Class Design
Class: CoffeeMachine

**Private Members**
- `waterLevel` → current water in machine
- `coffeeBeans` → current coffee beans
- `cashCollected` → money collected

**Public Functions**
- `getCash()` → check cash in machine
- `addWater(int)` → refill water
- `addBeans(int)` → refill beans
- `dispenseEspresso(double)` → dispense espresso if conditions are met

## Real-Life Application
- Models a real coffee machine
- Demonstrates OOP principles applied to practical systems

## Learning Outcome
- Learn how to model a real-world object using a class
- Practice managing internal resources inside a class
- Understand constructor and member function usage
